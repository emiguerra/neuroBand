# neuroBand
curso especialización 
- El taekwondo se transformaría en una disciplina de contacto donde la potencia y la velocidad de las patadas serían potenciadas por biotecnología
- El uso de cabezales sensorizados inteligentes permitiría no solo contabilizar puntos, sino también medir el impacto, detectar riesgos de conmoción y generar datos biométricos en tiempo real
<img width="1098" height="530" alt="Screenshot 2026-01-15 at 10 06 26 PM" src="https://github.com/user-attachments/assets/870fa099-863b-4aff-9636-401e3e0a11be" />

- Cuando la cabeza recibe un golpe, no solo importa la fuerza en línea recta, sino también qué tan rápido gira el cerebro dentro del cráneo
- Cuanto más rápido gire la cabeza tras un impacto, mayor es el riesgo de daño cerebral, y el casco inteligente puede detectar esos umbrales para avisar antes de que sea peligroso

## BOM
→ Acelerómetro triaxial: mide aceleración lineal.

→ Giroscopio: detecta rotación y movimientos peligrosos.

→ Sensor de presión: registra fuerza de cada impacto.

→ IMU 9DOF: integra datos para análisis completo de movimiento.
